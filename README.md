# WebChat

> 支持聊天记录, 重复登录判断

### 依赖

|PHP       |>= 5.5      |
| -------- | ---------- |
|Swoole    |>= 1.8      |
| -------- | ---------- |
|MsgPack   |>= 2.0      |
